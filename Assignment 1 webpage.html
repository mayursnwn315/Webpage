In this course of web designing, we will study HTML, CHTML, and Cascading Stylesheets (CSS). It is aimed at complete
beginners to the subject, and assumes you have no knowledge of creating web pages. Web design is the planning and creation of
websites. This includes the information architecture, user interface, site structure, navigation layout, colors, fonts, and imagery. All
of these are combined with the principles of design to create a website that meets the goals of owners and designers.

The webpage designing course includes
following topics:

  1. HTML - Hyper Text Markup Language
  2. DHTML - Dynamic Hyper Text Markup Language
  3. CSS - Cascading Style Sheet
  4. Scripting Language - Javascript

We also study the
following topics:

  • Frontpage
  • ASP
  • Dreamweaver
  • Webstandard
